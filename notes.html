CSS Position:

  1. static
    2. : same as defalt html position
  2. Relative
    3. : Move element from its real position relatively.
    - Other content will not be adjusted to fit into any gap left by the element.

    img{
    position: relative;
    left: 30px; }

  3. Absolute
    4. :Position element according(relative) to parent.
    - It affects the flow of elements.
    - parent should be relative position.
    - Absolute position takes element out of the normal postioning flow.
  4. Fixed
    5. :fixed to its position
